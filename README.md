# Freaky Foot Tours: Design Directions

Four full-page design-direction mockups for the freakyfoottours.com redesign,
each a complete homepage plus a tour page (The Original Flagstaff Ghost Tour),
built with real tour photography and verified review numbers only.

**View the gallery:** open `index.html`, or the hosted copy on GitHub Pages.

| # | Direction | Idea |
|---|---|---|
| 01 | The Night Marquee | Dark-first old-theater cinema; lantern-gold glow CTAs; real night photography |
| 02 | The Front Page | The town's broadsheet newspaper; press clippings as proof; tours as classified ads |
| 03 | The Poster House | The printed rack cards at full volume; illustrated poster hero; show-poster tour wall |
| 04 | Golden Hour | Premium photo-forward; dusk-blue and lantern-amber; floating booking card |

All four share the same conversion skeleton per the owner brief: sticky header
Book button, proof elements beside every CTA, button-styled links throughout,
persistent mobile book bar, and a final CTA section on every page.

Review copies only. The production site lives in the
[freakyfoottours](https://github.com/Eat-Sleep-Launch-Repeat/freakyfoottours) repo.
